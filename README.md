# 📌 SayWell: Dyslexic Aid Assistant

SayWell is an AI-powered web app that enhances reading and writing for individuals with dyslexia. It simplifies text, corrects spelling, and provides pronunciation guidance while ensuring an accessibility-first experience.

## ✨ Key Features  

- **AI Assistance:** Real-time text simplification, spelling correction, and pronunciation support.  
- **Custom Learning:** Personalized user profiles, adaptive difficulty, and a custom dictionary.  
- **Accessibility-Focused:** Dyslexia-friendly fonts, speech-to-text (STT), text-to-speech (TTS), and high-contrast UI.  
- **Secure Authentication:** Google and email/password login via Firebase.  

## 🛠️ Tech Stack  

- **Frontend:** React.js, Material-UI, Context API  
- **Backend:** Flask (Python), OpenAI API (via OpenRouter), PySpellChecker  
- **Database & Auth:** Firebase Firestore & Firebase Auth  

SayWell empowers users with an AI-driven, personalized learning environment that enhances literacy skills in a supportive way. 🚀
